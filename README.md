# File Organizer v2 – Smart Cross-Platform File Organizer

A cross-platform desktop app built with **Python** and **Tkinter** that lets you quickly organize messy folders with a live preview and one-click undo.

## Features

- Organize files by:
  - **File Type**
  - **Modified Date (YYYY-MM)**
  - **File Size** (Small / Medium / Large / Huge)
  - **Smart Categories** (Images, Videos, Audio, Documents, Archives, Code, etc.)
- Preview panel shows where every file will go *before* you run it
- Undo support – moves files back to their original locations
- Optional recursive mode to include subfolders
- Works on **macOS** and **Windows**
